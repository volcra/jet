Get Jet

Prerequisites

Java 7 is required to build and run Jet.

Get the code

Jet is in development and it may change continuosly until it reaches a GA version.

Currently the only way to get it downloading the source from GitHub.

git clone git://github.com/volcra/jet.git

Or just download the ZIP and exract in a directory of your choice.

Install

Gradle is used as build tool. The Gradle wrapper is provided for quick development. Gradle will be automatically downloaded and used to run the build.

Windows

In Windows the batch script is called gradlew.bat.

Unix

In Unix the batch script is gradlew.

Build

gradlew installApp

The installApp task is provided by the application plugin. This will create the executable with all the dependencies in the directory ${basedir}\build\install\jet.

Verify

Execute the command and you should expect the following output.

$>build\install\jet\bin\jet
usage: jet [options] [directories]
Options
 -h,--help          Print Help.
 -p,--port <port>   The Web Server port number. Defaults to 8080

Now you are ready to head to the Usage page.
