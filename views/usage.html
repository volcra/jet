Usage

How to get started

Getting Started

You need to first get Jet, go to the Download page for more information on how to install and set up Jet.

Getting Help

Run Jet passing -h or --help to get help.

$>jet --help
usage: jet [options] [directories]
Options
 -h,--help          Print Help.
 -p,--port <port>   The Web Server port number. Defaults to 8080

Command-Line Usage

Options

-p,--port

The Web Server port number, defaults to 8080.

$>jet -p 80
