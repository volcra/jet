Jet

Jetty Web Server made simple

Download GitHub Project

Download GitHub Project

Jet is a command line tool used to Run Jetty primary as a Web Server.

Quick Start

Command-Line

Download or build from source. For instructions go to Download.

Then run Jet on any directory to expose that directory as a Web Site.

$>jet .
